# How to set up
 - install uv [link](https://docs.astral.sh/uv/getting-started/installation/)
 - Clone this repo
 ```
    git clone https://github.com/ianYuinCellcubes/CC_CLUT_gen.git
 ```
 - move to CC_CLUT_gen
 ```
    cd CC_CLUT_gen
 ```
 - Configure using uv
 ```
    uv sync
    uv run main.py
 ```
